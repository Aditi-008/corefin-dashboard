# 💰 CoreFin – Finance Dashboard UI

A modern, responsive finance dashboard built using React and Tailwind CSS.  
This project focuses on clean UI/UX, state management, and interactive data visualization for tracking financial activity.

---

## 🚀 Features

### 📊 Dashboard Overview
- Summary cards: **Total Balance, Income, Expenses**
- 📈 Time-based visualization (Daily / Monthly / Yearly trends)
- 🥧 Category-wise spending breakdown (Pie/Doughnut chart)

---

### 💳 Transactions
- Add and delete transactions
- View details:
  - Date
  - Amount
  - Category
  - Type (Income / Expense)
- 🔍 Search functionality (category, amount, type, date)
- 🔃 Sorting (by date / amount)

---

### 👤 Role-Based UI
- **Admin**
  - Add/delete transactions
- **Viewer**
  - View-only access

---

### 🧠 Insights
- Highest spending category
- Monthly comparison (increase/decrease)
- Financial observation (income vs expense)

---

### 🎨 UI / UX
- 🌙 Dark / ☀️ Light mode
- Lavender premium theme
- Glassmorphism design
- Responsive layout
- Empty state handling

---

### ⚡ Enhancements
- 💾 LocalStorage persistence
- 📤 Export to CSV
- 📂 15 categories + custom category input
- Smooth transitions

---

## 🛠️ Tech Stack

- **React (Vite)**
- **Tailwind CSS**
- **Chart.js**
- **JavaScript (ES6+)**
- **LocalStorage**

---
